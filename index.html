Basement Builder

Basement Builder is a stage editor for Binding of Isaac: Rebirth. It allows you to modify and add rooms in the game. Basement Builder is forked from Basement Renovator made by Tempus.

How to run

Basement Builder works on every OS where Python runs (Windows, Linux, Mac OS X and more!). Just install Python 3, PyQt 5 and run this command in the terminal/CMD: python BasementBuilder.py in the root directory of the repository.
